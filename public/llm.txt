Description:
PortalPay enables businesses to accept crypto and card payments via onramp with QR-code receipts, instant settlement, unified billing, and real-time analytics. White-label branding, POS/PMS integrations, and enterprise features are included at no monthly cost.

Key Features:
- QR code payments from POS receipts
- Multi-token support (USDC, USDT, ETH, cbBTC, cbXRP)
- Onramp card payments → auto-convert to crypto
- Analytics dashboard and real-time transaction tracking
- White-label branding and theme customization
- Token reserve ratios and automatic rotation
- PMS/POS integrations and split routing

## API Overview

Public endpoints (examples based on standard REST patterns):
- POST /api/receipts — Create itemized receipts for QR payment portals
- GET  /api/transactions — Query transaction logs for analytics
- GET  /api/settings — Retrieve site configuration (branding, tokens)
- POST /api/split/deploy — Deploy split payment contracts
- GET  /api/inventory — List inventory items
- POST /api/orders — Create new orders
- GET  /api/shop — Get shop configuration

Note: Admin and sensitive endpoints are access-controlled and excluded from indexing.
